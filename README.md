# digitalhype_back

## requirements
This Repo requires:

- NodeJS (> v18)
- NPM
- AWS SDK g installed
- AWS CLI g installed
- AWS account configured with admin privileges

## deplot in AWS

- `cdk bootstrap`: To prepare inital stack.
- `cdk synth`: To synth with AWS stacks.
- `cdk deploy --all`: To deploy all AWS stacks, output displays end-points.
